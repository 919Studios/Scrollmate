Scrollmate

scrollmate.js scrolls from one element to another when a trigger is clicked. The scroll animation time can quickly be slowed down or sped up.

IGNORE THIS - This Section is just for adding space

Step #1: Add the trigger

Add the class .ScrollmateTrigger to the element you want to trigger the scroll animation.

IGNORE THIS - This Section is just for adding space

Step #2: Set the target area

Add the class .ScrollmateTriggerArea to an element (or area) you want to animate to.

IGNORE THIS - This Section is just for adding space

Step #3: Import the Javascript

Download Scrollmate JavaScript and import it at the bottom of your web page(s) with the code below:

<script src="/path/to/scrollmate.js"></script>

Modification Options:

Scrollmate Duration: The Scrollmate Duration (in milliseconds) is the time it takes to travel from .ScrollmateTrigger to the resting area .ScrollmateTriggerArea (demo uses: 1250 and 500).
